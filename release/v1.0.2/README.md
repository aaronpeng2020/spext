# Spext v1.0.2

基于 OpenAI Whisper API 的 Windows 语音输入工具

## 快速开始

1. 下载 `VoiceInput.exe`
2. 以管理员身份运行
3. 在设置中配置 OpenAI API 密钥
4. 按住热键（默认 F3）开始录音，松开后自动转换为文字

## v1.0.2 重要更新

- **API密钥保护**：彻底解决了 API 密钥丢失问题

## 系统要求

- Windows 10 或更高版本
- 管理员权限
- OpenAI API 密钥
- .NET Core 3.1 运行时（已包含在自包含版本中）

## 注意事项

- 首次运行请确保以管理员身份启动
- 如遇到输入问题，程序会自动选择最适合的输入方式
- 支持代理配置，可在设置中配置 HTTP 代理
